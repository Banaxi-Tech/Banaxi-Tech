# Hi, I'm Banaxi  👋

I build AI developer tools, low-level experiments, and unusual software projects.

My main project is **Banana Code**, a terminal-native AI coding assistant focused on fast developer workflows, multi-provider model support, project context, and tool calling.

## Featured projects

### [Banana Code](https://github.com/Banaxi-Tech/Banana-Code)

A terminal-native AI coding assistant.

It supports multiple AI providers, tool calling, persistent chats, workspace context, behavior modes, skills, memory, and safety controls.

Website: [bananacode.sh](https://bananacode.sh)

### [Banana Code Studio](https://github.com/Banaxi-Tech/Banana-Code-Studio)

A desktop GUI for Banana Code.

Built for using Banana Code in a more visual workflow outside the terminal.

### [BananaOS](https://github.com/Banaxi-Tech/BananaOS)

An experimental operating-system project written in C and assembly.

Focus areas include low-level graphics, boot/runtime experiments, terminal behavior, and systems programming.

### [v86-nx](https://github.com/Banaxi-Tech/v86-nx)

Experiments around browser-based x86 virtualization.
